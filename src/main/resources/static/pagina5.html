AMD Radeon Software



AMD Adrenalin


El nuevo Software de AMD permite a los usuarios que dispongan de una tarjeta grafica AMD poder controlar y hacer funciones directas desde el nuevo software el cual facilita a los usuario en su navegacion y poder sacar el maximo provecho de sus graficas con un software el cua se encuentra disponible para las plataformas de Windows 10, Windows 11 y Linux.


Caracteristicas de Adrenalin


Tuning


El nuevo software nos permite poder modificar parametros de nuestra tarjeta grafica en funcion de aumentar un rendimiento extra a la misma. El realizar estas preconfiguraciones, debemos de tener en cuenta que son configuraciones las cuales modificaran las revoluciones de los ventiladores, el voltaje en la que trabaja la grafica y en otros casos la memoria internada de las misma, etc.


Streaming


El Software agrega una nueva funcionalidad para aquellos usuarios que utilizan alguna plataforma de Streaming como lo es Twitch, Youtube, Facebook, etc. Ahora con este nuevo apartado podemos configurar parametro a nuestro gusto como lo pueden ser el apartado del microfono, proyectar el videojuego que estemos ejecutando, configurar nuestras camaras entre otras funciones.


AMD Link


Para Aquellos usuarios los cuales el gaming es su estilo de vida, el software de Adrenalin nos permitira controlar nuestro pc a distancia. Este contiende una interfaz siempre y sencilla para poder acceder a este apartado desde nuestros dispositivos moviles compatibles con la aplicacion, ya sea smartphones, Tables o incluso Smart TV que soporte dicho apartado.


Captura multimedia


Ahora realizar capturar asi como ver nuestros Replay y compartirlos sera mas facil con este apartado, ya que Adrenalin cuenta con un sistema el cual permite poder compartir las grabaciones o capturas que nosotros queramos en nuestras redes sociales de una forma directa y en buena calidad aprovechando cada funcion como lo es el Screenshot, Gif, Replay y Record Video.
